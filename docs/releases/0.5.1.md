# FreshSense AI 0.5.1 Public Beta

FreshSense 0.5.1 brings the redesigned product experience to the actual Windows
desktop application. It keeps the same validated on-device model pipeline and
privacy behavior while unifying the downloadable app with the newer FreshSense
visual system.

## Download

Download all three release files from the GitHub Release:

- `FreshSenseAI-Setup-0.5.1.exe`
- `FreshSenseAI-Setup-0.5.1.exe.sha256`
- `FreshSenseAI-Release-0.5.1.json`

Verify the installer before running it:

```powershell
Get-FileHash .\FreshSenseAI-Setup-0.5.1.exe -Algorithm SHA256
```

## What changed

- Rebuilt the PySide6 Windows interface around the redesigned FreshSense brand,
  editorial hero, scanner hierarchy, privacy messaging, and five-stage analysis
  route.
- Added explicit empty, loading, selected-photo, accepted, withheld, error, and
  high-risk result states.
- Preserved the supported-input gate, DenseNet201 classifier, local semantic
  retrieval, optional reasoning, Grad-CAM, metadata-only history, and incorrect-
  result reporting behavior.
- Added a reproducible, cursor-free screenshot workflow that runs the real model
  and captures the rendered Windows UI.
- Updated README screenshots and the 30-second walkthrough to show the desktop
  application users actually download.

## Privacy and safety

FreshSense analyzes selected photos locally and does not create an application
copy or automatically upload the image. It provides visual decision support
only and cannot prove that food is safe to eat or detect internal spoilage,
contamination, odor, texture, pathogens, or chemical hazards.

This beta is not Authenticode signed. Windows may show an unrecognized publisher
warning. The SHA-256 checksum verifies file integrity but not publisher identity.
