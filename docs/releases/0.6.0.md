# FreshSense AI 0.6.0 Public Beta

FreshSense 0.6.0 expands the Windows beta from three fruits and six freshness
classes to six produce types and 12 classes. The release adds fresh and rotten
mango, tomato, and pear while retaining apple, banana, and orange support.

## Download

Download all three release files from the GitHub Release:

- `FreshSenseAI-Setup-0.6.0.exe`
- `FreshSenseAI-Setup-0.6.0.exe.sha256`
- `FreshSenseAI-Release-0.6.0.json`

Verify the installer before running it:

```powershell
Get-FileHash .\FreshSenseAI-Setup-0.6.0.exe -Algorithm SHA256
```

## What changed

- Added fresh and rotten mango, tomato, and pear classifications.
- Trained a 12-class head on an ImageNet-pretrained DenseNet201 visual backbone.
- Prepared a group-isolated dataset manifest so crops from one source image do
  not cross training, validation, and test splits.
- Recalibrated the model-bound supported-input gate for all 12 classes.
- Added storage and visible-spoilage guidance for mango, tomato, and pear.
- Added reproducible WSL2 GPU training, evaluation, artifact verification, and
  all-class Windows smoke-test scripts.
- Cryptographically associated the model, gate, catalog, dataset manifest, and
  evaluation report in the release artifact manifest.

## Development evaluation

The grouped development test contains 4,043 supported images. Before
withholding, the classifier reached 98.54% accuracy. After the supported-input
and confidence policy, accepted predictions reached 99.16% selective accuracy
at 88.67% coverage. Nineteen of 2,158 rotten test images were accepted as fresh
(0.88%). All 192 synthetic unsupported images were rejected.

These are development results from curated/source-export data, not an
independently photographed store benchmark. They must not be described as
general real-world accuracy.

## Privacy and safety

FreshSense analyzes selected photos locally and does not create an application
copy or automatically upload the image. It provides visual decision support
only and cannot prove that food is safe to eat or detect internal spoilage,
contamination, odor, texture, pathogens, or chemical hazards.

This beta is not Authenticode signed. Windows may show an unrecognized publisher
warning. The SHA-256 checksum verifies file integrity but not publisher identity.
